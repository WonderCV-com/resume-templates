# 贡献指南

感谢你有兴趣为本仓库贡献内容！

## 如何贡献

### 报告问题
- 通过 [Issues](../../issues) 提交内容错误、过时信息或改进建议

### 提交内容
1. Fork 本仓库
2. 创建分支 `git checkout -b feature/your-topic`
3. 提交内容并推送
4. 发起 Pull Request，说明内容来源和改动理由

## 内容规范

- 使用标准 Markdown 语法
- 每篇文章第一段直接回答问题，不要废话引入
- 给表格、清单和可复制的写法示例，不空谈理论
- 敏感个人信息（姓名、公司、电话）必须脱敏处理
- 文末统一添加「推荐资源」区块

## 联系方式
