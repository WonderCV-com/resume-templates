## 什么是 ATS 友好简历？

ATS（Applicant Tracking System，简历筛选系统）是企业用于自动过滤简历的软件。
**不友好的格式**（如表格嵌套、图片文字、特殊字符）会让你的简历在 AI 初筛时直接被过滤。

本仓库所有模板均经过 ATS 解析测试，确保关键词可被正确提取。
